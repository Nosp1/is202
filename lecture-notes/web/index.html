IS-202 Lecture Notes

Introduction

Java servlets is a simple platform for building web applications. Most real life project will choose some kind of framework built on top of servlets, rather than using servlets directly. However, we have chosen to teach the use of plain servlets in the course, because we belive that understanding the basic platform makes it easier to understand the more advanced frameworks later.

The course will advance in small steps. Each step will introduce one feature of the servlet platform, (As far as possible. Some features have strong mutual dependencies). The lecture topics will be:
