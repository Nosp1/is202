Secure Webapp

This project demonstrates BASIC authentication, and the UserDatabaseRealm. These are not suited for production, but they are quick and easy to set up, and they can be just as easily replaced in a production environment so they are a good choice for development.

The test servlet will handle any url that ends in .a, .s, or .t, but login as sysadm, student, or teacher is required. Open CATALINA_HOME/conf/tomcat-users.xml to see the defined users.
