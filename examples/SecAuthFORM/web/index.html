Secure Webapp
Form authentication

This project started as a clone of the SecAuthBASIC project. There are just a few lines of new code, and a few new html pages.

The main difference is that this project uses FORM based authentication. That is, rather than displaying a dialog, the system will display a page with an html form where the user can enter her credentials when she wants to log in. Furthermore, there is a logout button, so you can log out, and login as another user, without having to restart the browser.
